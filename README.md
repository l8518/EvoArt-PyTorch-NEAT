# EvoArt-PyTorch-NEAT

This repository adapts Uber's [PyTorch-NEAT](https://github.com/uber-research/PyTorch-NEAT) implementation built on [Python-NEAT](https://github.com/CodeReclaimers/neat-python) to generate
evolutionary art.

## Audio Preprocessor

Requires a loopback device otherwise it will use other default input device (probably your microphone).

### Loopback Software

- Windows:
  - `https://www.vb-audio.com/Cable/` install and reroute your output
- Linux:
  - Have not checked
- MacOs
  - Have not checked
